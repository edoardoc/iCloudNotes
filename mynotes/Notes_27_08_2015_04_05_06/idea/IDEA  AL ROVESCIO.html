IDEA "AL ROVESCIO"

scrivi una app che usa le seguenti funzionalità: 
(al posto di scegliere un idea e poi risolverne le specifiche, si parte dalle specifiche che siamo in grado di risolvere per arrivare ad un possibile set di applicazioni)

1. accede ad un server remoto, anche in maniera intensiva (sia di traffico che di CPU)

2. accesso tramite web service java (molto semplice da realizzare, si sposta il core dell'applicativo sul server)

3. possibilità di non usare server pubblici ma dare la possibilità ad ogni utente di "tirarsi su" il proprio (alla fine ogni mac con java + jetty + vpn è un server personale)


idee:

- archiviazione e sync privato: si sfrutta 3 per avere spazio illimitato sul proprio desktop da usare nel device iOS (ad es. iPad che visualizza foto già preorganizzate, dropbox privato, ecc.)

- stream bouncer: tramite 3 e 2 si proxano richieste di stream che vengono negate da un paese per essere "rimbalzate" da un paese che le permette -> tutto personale, tutto privato, altrimenti si rischia oscurazione.
